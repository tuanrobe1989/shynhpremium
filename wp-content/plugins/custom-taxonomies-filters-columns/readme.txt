Simple Admin UI for creating Custom Post Types Taxonomies Filters and Sortable Columns in edit admin screen.


== Description ==

CTFC provides taxonomy filters and/or sortable taxonomy columns for any CPT taxonomies in Edit Screen with only 1 simple admin page.

CTFC is a good add-on for themes or plugins that ease Custom Post Types and Custom Taxonomies Creation as, sometimes, they don't provide Custom Taxonomies Filters or Custom Taxonomies Sortable Columns for the registerd CTP.

As nothing can prevent you from extravagant multi categorization, the taxonomy column sorting works more relevantly with mono categorized contents.

== Installation ==
= Admin Installer via search =
1. Visit the Add New plugin screen and search for \"Custom Taxonomies Filters & Columns\".
2. Click the \"Install Now\" button.
3. Activate the plugin.
4. Navigate to the \"CTFC\" Menu.

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the \"Upload Plugin\" button.
2. Click the \"Browse...\" button and select zip file from your computer.
3. Click \"Install Now\" button.
4. Once done uploading, activate Custom Taxonomies Filters & Columns.

= Manual =
1. Upload the Custom Taxonomies Filters & Columns folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the \"CTFC\" Menu.
4. Windows Users : first globaly activate the functionality for type post type you want, save, then check the filter and/or column option
